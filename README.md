# Retail-Sales-SQL
# 🛒 Retail Sales Data Analysis using SQL & Python

This project analyzes retail sales data using SQL and Python. It focuses on data cleaning, exploration, and deriving business insights from transactional sales data.

## 📁 Dataset Columns

- Transaction ID
- Sale Date
- Sale Time
- Customer ID
- Gender
- Age
- Category
- Quantity
- Price per unit
- COGS
- Total Sale

## 🧹 Data Cleaning (in SQL)

- Removed negative prices and sales
- Ensured date and time formats were correct
- Cleaned missing or invalid gender/age entries

## 📊 Analysis Performed

- Total and average sales
- Profit by category
- Sales trends over time
- High-value customers
- Sales by time of day, weekday, gender, and age group

## 📌 Tools Used

- **SQL:** PostgreSQL
- **Python:** Pandas, Matplotlib, Seaborn, SQLAlchemy
- **Jupyter Notebook:** Visualizations & reporting

## 📈 Visualizations

- Sales by time of day (bar chart)
- Daily sales trend (line chart)
- Customer segments (pie chart)
- Category-wise profit (bar chart)


## 📚 Learning Outcomes

- End-to-end project workflow from raw data to business insights
- Hands-on experience with SQL and Python integration
- Visual storytelling using data
